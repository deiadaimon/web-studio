<h1>Web Studio</h1>
<p>Web Studio is an example of adaptive website developed for all main types of devices: smartphones, tablets and desktops.</p>
<p>Its markup and styles are written according to HTML5 and CSS3 standards with attention paid to semantics of the content. The code passed the validator successfully, no errors or warnings detected. In addition, the website is optimized to be compatible with search engines and increase it's accessability.</p>
<p>The images are compressed but still maintain the high quality and provided in additional high resolution adapted for retina displays.</p>
