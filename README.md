<h1>WebStudio</h1>

<p>Web Studio is an example of a simple yet effective <b>landing page</b>. It is an
<b>adaptive website</b> developed to provide an optimal experience across all major
device types: smartphones, tablets, and desktops.</p>

<p>The codebase adheres strictly to <b>HTML5</b> and <b>CSS3</b> standards, prioritizing <b>semantic
markup</b> and passing code validation with zero errors or warnings. The
architecture follows <b>Concentric CSS</b> methodology, ensuring the stylesheets are
highly organized, maintainable, and scalable. By leveraging <b>CSS Custom
Properties</b> (variables) for design tokens and <a href="https://cdnjs.com/libraries/modern-normalize" target="_blank" rel="noopener noreferrer">modern-normalize</a>, the project maintains a consistent visual identity and flawless cross-browser performance.</p>

<p>Beyond clean code, the site is fine-tuned for <b>search engine compatibility</b> and
improved <b>accessibility</b>, featuring performance-optimized, <b>high-resolution images</b>
adapted for Retina displays.</p>

<h2>Tech Stack & Tools</h2>
<ul>
<li><b>Core:</b> HTML5 (semantic markup), CSS3 (modern layouts), JavaScript</li>
<li><b>Styling & Architecture:</b> CSS Variables for centralized theme management, Concentric CSS for property organization</li>
<li><b>Consistency:</b> <a href="https://cdnjs.com/libraries/modern-normalize" target="_blank" rel="noopener noreferrer">modern-normalize</a> for cross-browser stability</li>
<li><b>Strategy:</b> Mobile-First Design, Adaptive Web Design</li>
<li><b>Build System:</b> Vite (<a href="https://github.com/goitacademy/vanilla-app-template" target="_blank" rel="noopener noreferrer">Vanilla App Template</a>)</li>
<li><b>Optimization:</b> Image minification, high-density imagery Retina support, SEO best practices, Web Accessibility</li>
<li><b>Deployment & Validation:</b> GitHub Pages, <a href="https://www.w3.org/developers/tools/" target="_blank" rel="noopener noreferrer">W3C Markup Validation Service</a></li>
</ul>
